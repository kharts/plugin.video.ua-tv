# UA-TV (Kodi/XBMC plugin) 
UA-TV: Kodi/XBMC plugin for watching Ukrainian TV Channels.  
Uses live streams from official pages of channels.  
Supported channels:  
- 5 channel
- Business TV
- Espreso TV  
- 24 channel  
- Hromadske TV  
- 112 channel  
- ICTV  
- NewsOne  
- UA|TV  
- UBR
- Ukraina
- UKRLIFE.TV
